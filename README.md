# Agent Ops Prompt Generator

A free, local-first browser and CLI tool that generates Korean prompts for AI coding agents during operations work.

Use it for:

- Incident triage
- Deployment smoke checks
- Code review
- Data correction
- Handoff summaries

## CLI

Run from GitHub:

```bash
npx --yes github:mysubb01/agent-ops-prompt-generator incident --target admin-api --risk high --context "500 errors after deploy"
```

Print the paid template-pack purchase request link:

```bash
npx --yes github:mysubb01/agent-ops-prompt-generator --buy
```

## Privacy

- No login
- No tracking
- No network requests from the tool
- Everything runs in the browser or local terminal
